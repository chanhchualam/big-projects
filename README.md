## 📌 Tổng quan (Overview)
Dự án được xây dựng trên nền tảng **ASP.NET Core MVC**, giải quyết bài toán [A, B, C...].

## 📊 Kỹ năng Business Analyst (BA Skills)
Trong dự án này, tôi đã thực hiện:
- **Phân tích yêu cầu:** Chuyển đổi nhu cầu người dùng thành các tính năng kỹ thuật.
- **Thiết kế sơ đồ:** Sử dụng UML (Use Case, Activity Diagram) để mô tả luồng nghiệp vụ.
- **Thiết kế Database:** Xây dựng sơ đồ ERD (Entity Relationship Diagram) đảm bảo tính toàn vẹn dữ liệu.

## 🛠 Công nghệ & Công cụ
- **Backend:** .NET Core, C#
- **Database:** SQL Server
- **Công cụ phân tích:** Draw.io / PowerDesigner
