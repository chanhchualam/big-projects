Đây là bài tập trên SQL Server, không gọi từ Flask. Trigger được tạo trong SSMS trên instance Miền Nam, database CSGT_MIENNAM.

Yêu cầu đề bài

  • Sự kiện: DELETE trên dbo.LoiViPham (xóa một mã lỗi khỏi danh mục).
  • Điều kiện: chỉ được xóa nếu MaLoi đó chưa xuất hiện trong ChiTietPhat ở bất kỳ miền nào (lưu vết lịch sử).
  • Kỹ thuật: trigger trên Nam phải kiểm tra ChiTietPhat tại Nam (local) + Trung (LINK_TRUNG) + Bắc (LINK_BAC).

Thứ tự triển khai

  1. Trên instance Miền Nam: tạo linked server LINK_TRUNG, LINK_BAC — xem khối comment trong sql/13_setup_linked_servers.sql (phần Miền Nam). Script có IF NOT EXISTS để chạy lại an toàn.
  2. Mở sql/15_cau_6_trigger_loi_vi_pham_mien_nam.sql (hoặc khối "CÂU 6" trong sql/12_bai_tap_cau_3_den_7_phan_tan.sql — cùng nội dung, không phải file lồng nhau).
  3. Execute trên database CSGT_MIENNAM.
  4. Kiểm tra: SELECT name FROM sys.triggers WHERE parent_id = OBJECT_ID(N'dbo.LoiViPham');

Hành vi

Nếu MaLoi vẫn còn trong ChiTietPhat ở một trong ba nơi → RAISERROR và ROLLBACK TRANSACTION → lệnh DELETE bị hủy.
